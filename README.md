# 📧 Mail - Email Client

A simple single-page email client built with JavaScript and Django as part of CS50’s Web Programming with Python and JavaScript.

---

## Features

- Send and receive emails  
- Inbox, Sent, and Archive mailboxes  
- Read email details  
- Archive and unarchive emails  
- Reply with pre-filled fields  

---

## Project Structure

- `inbox.js` : Handles all frontend logic  
- `inbox.html` : Main UI layout  
- `styles.css` : Styling  

---

## How to Run

1. Run migrations  

python manage.py makemigrations
python manage.py migrate

2. Start server  

python manage.py runserver

3. Open in browser:  

http://127.0.0.1:8000/

---

## Notes

- Built as a single-page app (no page reloads)  
- Uses Fetch API for backend communication  
- Emails are stored in local database
